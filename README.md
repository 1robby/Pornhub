## 用法简介

- ```git clone https://github.com/killmymates/Pornhub ```
- ```cd Pornhub && python crawler.py run list```
- 待程序运行完毕， 会在webm文件下download两页的webm缩略图，对应名称为详细页面的URL后缀
- 把后缀放到download.txt里，再运行python crawler.py run, 在MP4文件里可看到下载好的MP4文件
